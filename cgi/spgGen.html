About:

SpgGen is an algorithm developed by Patrick Avery. SpgGen generates random crystals of specific spacegroups and allows the user to specify a lot of information such as minimum interatomic distances, volume constraints, etc. The source code is available under the "New" BSD open source license and may be viewed at its Github repository.
